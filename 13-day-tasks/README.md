# Week 3 Day 3 – Port Scanner

## Objective

Develop a Python program that scans common TCP ports on a target IP address using Python sockets.

## Features

* Accepts a target IP address or hostname.
* Scans commonly used TCP ports.
* Identifies open ports.
* Displays the associated network service for each open port.
* Measures the total scan time.
* Saves the scan results to `results.txt`.
* Handles common errors such as invalid IP addresses, connection timeouts, and interrupted scans.

## Technologies Used

* Python
* Socket Module
* Time Module
* Datetime Module

## Outcome

Successfully developed a TCP Port Scanner capable of identifying common open ports on a target system using Python sockets while generating a formatted scan report.
