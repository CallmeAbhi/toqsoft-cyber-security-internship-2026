# Week 3 Day 4 – Password Strength Checker

## Objective

Develop a Python program to evaluate the strength of a password based on common security criteria.

## Features

* Accepts a password from the user.
* Evaluates password length.
* Checks for:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* Calculates a strength score.
* Assigns a password rating.
* Provides suggestions to improve weak passwords.

## Technologies Used

* Python
* String Module
* Match-Case Statements

## Outcome

Successfully developed a Password Strength Checker capable of evaluating password security, assigning a strength rating, and providing recommendations to improve password quality.
