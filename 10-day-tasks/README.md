# Week 2 Day 5 – Top 5 Web Security Risks Presentation

## Topic

Top 5 Web Security Risks

## Reference

The presentation is based on the OWASP Top 10:2025 documentation and covers the following risks:

* A01:2025 – Broken Access Control
* A02:2025 – Security Misconfiguration
* A03:2025 – Software Supply Chain Failures
* A04:2025 – Cryptographic Failures
* A05:2025 – Injection

## Contents

The presentation explains:

* Definition of each security risk
* Why it occurs
* Example attack scenarios
* Business impact
* Prevention methods
* Comparison of all five risks
* Common security best practices

## Outcome

Prepared a professional presentation summarizing the Top 5 OWASP Web Security Risks using the official OWASP Top 10:2025 guidance.
