# SQL Injection and XSS Demonstration Lab

## Purpose

A local educational lab created to understand common web application vulnerabilities and their mitigations.

## Components

### SQL Injection Lab
- Express.js
- SQLite3
- Vulnerable query construction
- Parameterized query mitigation

### Reflected XSS Lab
- Express.js
- Direct HTML injection demonstration
- Output encoding mitigation

### XSS Presentation Simulator
- Pure HTML/CSS/JavaScript
- Demonstrates innerHTML vs textContent
- Built for educational presentation purposes

## Security Notice

This project is intended for local educational use only and must not be exposed to public networks.
